-You must change variables DISCORD_TOKEN  + GUILD_ID in the .env file for the bot to work.

List of commands:

Send a DM to me - allows a discord user to create a ANONYMOUS ticket with the message that was sent to me, which will be analyzed by the admins
/ticket 'msg' - allows a discord user to create a ticket with 'msg' as the reason
/respond - (admin) allows admin to respond to a ticket
/close - (admin) close a ticket. This deletes the channel after 5 seconds
